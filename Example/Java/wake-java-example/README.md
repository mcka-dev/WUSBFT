# wake-java-example

## Programming language
Java 8

## Dependencies
- Oracle Java 8 or OpenJDK 8
- JavaFx 8 or OpenJFX 8
- FTDI D2XX Driver

## Building Project
Apache Maven

## The software runs on operating systems 
- Windows XP/Vista/7/8/8.1/10 (x86/x86-64)
- Linux Debian/Ubuntu/OpenSuse/Fedora (x86/x86-64)
